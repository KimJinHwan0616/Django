### home/urls.py ( 생성 )
```angular2html
from django.urls import path
from . import views

urlpatterns = [
    path('', views.HomeView.as_view(), name='home'),    # 홈 VIEW 연결
]
```

### templates/layout/base.html ( 생성 )
```angular2html
<!DOCTYPE html>
<html lang="ko">
<head>
    {% load static %}
    <meta charset="UTF-8">
    <link rel="stylesheet" href="{% static '/css/스타일_이름.css' %}">
                            ...
    
    <title>{% block title %}제목{% endblock %}</title>
    
    <style>
        {% block style %}
            CSS 코드
        {% endblock %}
    </style>
    
</head>

<body>

    {% include 'layout/header.html' %}   <!-- 헤더 불러오기 -->
    
    {% block 레이아웃_태그 %}
        HTML 코드
    {% endblock %}
    
    {% include 'layout/footer.html' %}   <!-- 푸터 불러오기 -->
    
    {% block script %}
        자바스크립트 코드
    {% endblock %}

</body>
</html>
```
